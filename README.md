#SETUP/NOTES
1. This Angular project was made using Angular v21 so make sure that your Angular is running on that environment.
2. The built-in port that was used for making http request to the API endpoint to the server is set to 44313.

#NOTE:
- If you cannot run the API or your Visual Studio in IIS or 44313 PORT. You can just manually change the port number of the API endpoint in the environment.ts
  of the Angular project in Visual Studio Code.
